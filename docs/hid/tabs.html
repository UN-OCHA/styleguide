Tabs

Responsive tabs - changes to accordion on small screens.

Sass: common/_tabs.scss

Options:

  • Tab width - by default the tabs have width: auto. Add classes 'tab-x' to set the number of tabs and have them take up the full width (eg, if set to 2 each tab will be 50% width), available numbers of tabs are 2, 3 and 4.
  • tabsActive - false by default. This closes the first tab initially on small screens. Set to true to have it be open on page load.

Javascript (in controller)

{% highlight html %} $scope.tabsActive = false; $scope.activeTab = 1; $scope.toggleTabs = function (tabName) { $scope.activeTab = tabName; $scope.tabsActive = true; }; {% endhighlight %}
