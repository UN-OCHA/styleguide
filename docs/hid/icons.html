Icons

HID uses a sub set of the main icon library generated with IcoMoon.

The icon font and selection.json file are in the repository - https://github.com/UN-OCHA/hid_app2/tree/dev/src/assets/fonts/icons

Sass: common/_icons.scss

Usage:

Use the icon directive, specifiying the name of the icon

If the icon is not accompanied by visible text you should give text for screen readers in the 'text' parameter.
