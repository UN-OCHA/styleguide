Form sections

To divide a form into section wrap each section in a div or fieldset with class form-section.

For smaller spacing add the modifier class form-section--compact.

Sections can have headings, using class form-section__heading and sub-headings, using class form-section__sub-heading. These can be applied to heading or legend elements as appropriate.
