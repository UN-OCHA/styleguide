Common Design

The Common Design project aims to provide a visual framework that will help:

  • Navigate seamlessly accross OCHA services and platforms
  • Improve users' understanding of all digital services provided by OCHA and their relations to each other
  • Improve user experience, efficiency and satisfaction
  • Create a systematic structure that allows future additions with minimal design support
