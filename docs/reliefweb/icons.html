ReliefWeb / Icons

ReliefWeb uses a sub set of the main icon library as an svg sprite, with png fallback for IE8.

Usage:

In HTML: {% highlight html %}{% endhighlight %}

In Sass: {% highlight html %} @include background-svg('icon-sprite'); background-position: 0 -XXpx; background-repeat: no-repeat; {% endhighlight %}

Sass file: components/_icons.scss
