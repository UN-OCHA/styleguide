ReliefWeb / Colours

Use the contextual colours (text, background, border) rather than the palette colours directly.

Use a colour contrast checker to confirm that your colour combinations are accessible - http://webaim.org/resources/contrastchecker/.

Sass file: variables/_colors.scss

Colour palette

  • $red: ##F15A46; (Note: this red is not accessible with light or dark text, use dark red alternative with text.)
  • $dark-red: #cb250f;
  • $blue: #07719B;
  • $purple: #5C6FA6;
  • $dark-grey: #333;
  • $mid-grey: #555;
  • $mid-light-grey: #d4d4d4;
  • $light-grey: #f4f4f0;
  • $lighter-grey: #fcfcfc;

Text colours

  • $text-dark: $mid-grey;
  • $heading-text-color: $dark-grey;
  • $link-text-color: $blue;
  • $link-text-active-color: darken($link-text-color, 12%);

Background colours

  • $light-bg: $lighter-grey;
  • $block-bg: $light-grey;
  • $highlight-bg: $dark-red;

Border colours

  • $light-border: $light-grey;
  • $mid-border: $mid-light-grey;

Header colours

  • $mobile-header-color: $purple;
  • $header-tab-focus: #e9e9e5;
  • $header-box-shadow-color: #a3a3a3;
  • $header-search-border: #e4e4e2;
  • $header-search-bg: #f9f9f8;
