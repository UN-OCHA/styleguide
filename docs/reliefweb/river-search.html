ReliefWeb / River Search

Reduce screen width to see mobile version.

Requires responsive-search.js and simpledatepicker.js

Sass file: modules/reliefweb_rivers/sass/_search.scss

How to search

Refine the results by adding specific criteria.

  • Select an operator. Use "WITH" and "WITHOUT" operators to create a new group.
  • Select a category, then start typing a word to see the suggestions.
  • Click or press enter to select the suggestion.
  • Once finished, click "Apply options" to launch the search.
  • Optional: click on an operator to choose a different one.
  • Optional: drag and drop criteria to reorganize the selection.
  • Optional: press "backspace" to remove the last criterion.
  • Optional: double click on a criterion to remove it.
