Style guide for the OCHA Basic Drupal starter theme: https://github.com/UN-OCHA/ocha_basic

The Sass structure can also be used on non-Drupal sites.

Base styles and helpers

Sass file: /sass/base/_grid.scss

Sass file: /sass/base/_helpers.scss

Extras

These component styles aren't included in the OCHA Basic theme, which is kept minimal, but can be added by downloading the .scss files and including in your version of the theme.
