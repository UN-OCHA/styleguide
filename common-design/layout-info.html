Common Design / Layout

A minimal layout

The Common Design doesn't affect the internal layout of pages or a grid, however to ensure consistency across sites it does recommend these two rules are followed:

Maximum page width & padding

In the Common Design the maximum page width is 1400px and the maxiumum content width is 1220px, see below for details.

A maximum page width is necessary on responsive sites to prevent the layout breaking down and lines of text becoming to long to comfortably read on large screens. Although there is no industry standard page width, from reviewing several well-known websites the trend is to set it between 1000px and 1400px. Most OCHA sites also come within this range.

Setting page width and padding ensures that the design retains consistency across different sites.

Mobile first, fluid responsive layout

A fluid, as opposed to adaptive, responsive layout is favoured as this allows us to utilise a maximum of screen real estate (up to the maximum width). It also helps to reinforce the design consistency if this is applied on all common design sites.
