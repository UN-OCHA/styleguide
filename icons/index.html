Icon library

How to create an icon set:

It is recommended that you only include icons that you need in your project to reduce file size. To this you should create a subset of the unocha-icons set:

  1. Download the unocha-icons selection.json file
  2. Go to IcoMoon - https://icomoon.io/app/
  3. Import the icons - click 'Import icons', select selection.json
  4. Click yes if asked if you would like to load all the settings stored in your selection file.
  5. Remove the icons you do not want in your icon set.
  6. Generate and then download the font. (Alternatively, the icons can be downloaded as svgs and used instead of an icon font.)

See the IcoMoon documentation for full details of how to implement an icon font.
